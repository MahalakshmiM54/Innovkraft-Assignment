# Innovkraft Assignment

## Features
- Home Page
- User Registration
- Login
- Membership Pricing
- My Account
- WhatsApp Integration

## Technologies Used
- WordPress
- Elementor
- XAMPP
- MySQL

## Database
Import the file `wordpress_db.sql` into phpMyAdmin.

## Files Included
- Database Export (.sql)
- WordPress Content Export (.xml)
- Screenshots

## Local Setup
1. Import database.
2. Copy project to XAMPP htdocs.
3. Start Apache and MySQL.
4. Open localhost URL.

## Demo Link
Local Environment:
http://localhost:8081/wordpress-7.0/wordpress/

Note: This project runs locally using XAMPP.

## Installation

1. Install XAMPP.
2. Copy project files into htdocs.
3. Create database named wordpress_db.
4. Import wordpress_db.sql into phpMyAdmin.
5. Start Apache and MySQL.
6. Open:
   http://localhost:8081/wordpress-7.0/wordpress/

## Notes
This project was developed using WordPress and Elementor.
Database and content export files are included in this repository for local setup.
